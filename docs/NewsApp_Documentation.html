1. Project Overview

News App is a modern Android application built using Kotlin and Jetpack Compose that allows users to browse, search, and save their favorite news articles. The app follows clean architecture principles with MVVM pattern for better code organization, testability, and maintainability.

Key Features

  • User Authentication: Login screen with email/mobile and password validation
  • News Feed: Display top headlines from NewsAPI
  • Search: Search news articles with debounced query
  • Favorites: Save and manage favorite articles locally
  • Article Details: View full article with favorite toggle

5. Data Layer

5.1 API Service (NewsApiService.kt)

Retrofit interface for communicating with NewsAPI.org

getTopHeadlines(country: String, apiKey: String): NewsResponse

Endpoint: GET /v2/top-headlines

Parameters: country (default: "us"), apiKey

Returns: NewsResponse with list of articles

searchNews(query: String, apiKey: String): NewsResponse

Endpoint: GET /v2/everything

Parameters: query (search term), apiKey

Returns: NewsResponse with matching articles

5.2 Local Database

Room database for persisting favorite articles locally.

FavoriteArticle Entity

Field Type Description
url String (PrimaryKey) Unique article URL
title String? Article headline
description String? Article summary
urlToImage String? Image URL
author String? Author name
publishedAt String? Publication date
content String? Article content
sourceName String? News source name
addedAt Long Timestamp when favorited

5.3 Repository (NewsRepository.kt)

Single source of truth that mediates between data sources.

getTopHeadlines(): Result<List<Article>>

Fetches top headlines from API and returns wrapped in Result sealed class.

searchNews(query: String): Result<List<Article>>

Searches articles matching the query from API.

getAllFavorites(): Flow<List<FavoriteArticle>>

Returns reactive stream of all favorite articles from local database.

toggleFavorite(article: Article): Boolean

Adds or removes article from favorites. Returns true if added, false if removed.

9. Functions Reference

LoginViewModel Functions

onEmailOrMobileChange(value: String)

Updates email/mobile field and clears any existing errors.

onPasswordChange(value: String)

Updates password field and clears any existing errors.

onSignInClick()

Validates inputs and checks credentials against hardcoded values.

isValidEmailOrMobile(input: String): Boolean

Validates if input matches email pattern or 10-digit mobile number.

HomeViewModel Functions

loadNews()

Fetches top headlines and updates UI state. Filters out removed articles.

observeFavorites()

Collects favorites flow and maps FavoriteArticle to Article.

refresh()

Sets refreshing state and reloads news.

SearchViewModel Functions

onQueryChange(query: String)

Updates query state and triggers debounced search after 500ms.

searchNews()

Executes search API call and updates results.

clearSearch()

Resets search state to initial values.

FavoritesViewModel Functions

observeFavorites()

Collects favorites flow from repository.

removeFromFavorites(article: Article)

Removes article from favorites database.

NewsDetailsViewModel Functions

observeFavoriteStatus(url: String)

Observes if article is favorited and updates UI state.

toggleFavorite()

Adds or removes current article from favorites.

Repository Functions

getTopHeadlines(): Result<List<Article>>

Fetches headlines from API, returns Success or Error result.

searchNews(query: String): Result<List<Article>>

Searches articles from API with given query.

getAllFavorites(): Flow<List<FavoriteArticle>>

Returns reactive flow of all favorites sorted by addedAt DESC.

isFavorite(url: String): Flow<Boolean>

Returns flow indicating if article is favorited.

addToFavorites(article: Article)

Converts Article to FavoriteArticle and inserts into database.

removeFromFavorites(url: String)

Deletes favorite by URL from database.

toggleFavorite(article: Article): Boolean

Toggles favorite status. Returns true if added, false if removed.
